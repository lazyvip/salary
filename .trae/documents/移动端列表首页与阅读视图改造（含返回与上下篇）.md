## 目标
- 移动端首页仅显示总目录（列表），不显示正文与目录浮层。
- 点击进入阅读视图后：隐藏目录，仅在左上角显示“返回”按钮；底部增加“上一篇/下一篇”卡片并显示相邻标题。
- 刷新保留状态：使用 hash 路由 `#list` 与 `#post/<filename>`。

## 交互与路由
- 初始：若 hash 为 `#post/<filename>` 则进入阅读视图并加载文章；否则进入列表视图 `#list`。
- 列表→阅读：写入 `#post/<filename>`，切换到阅读视图并滚动置顶。
- 阅读→返回：写入 `#list`，切回列表视图并恢复列表滚动位置。
- 上一/下一篇：根据当前索引计算相邻文章，禁用越界；点击时更新 hash 并加载相邻文章。

## 页面结构调整
- index.html：
  1) 在阅读区顶部添加移动端返回按钮：`<button id="mobile-back" class="mobile-back">返回</button>`
  2) 在阅读区底部添加分页容器：
  ```
  <div id="mobile-pager" class="mobile-pager">
    <a id="pager-prev" class="pager-card pager-prev"><span class="label">上一篇</span><span id="pager-prev-title"></span></a>
    <a id="pager-next" class="pager-card pager-next"><span class="label">下一篇</span><span id="pager-next-title"></span></a>
  </div>
  ```

## 样式（移动端断点 <768px）
- 新增模式类：`.mode-list` 与 `.mode-reader`，挂在 `body` 上。
- `.mode-list`：
  - 列表全屏可滚动；隐藏阅读区与返回/分页；隐藏目录浮层与右下角“目录”按钮。
- `.mode-reader`：
  - 阅读区全屏；隐藏侧栏列表与目录浮层；固定顶部“返回”按钮；底部显示分页卡片。
- 分页样式参考 VitePress：左右卡片，显示相邻标题，虚线边与轻阴影，安全区内边距 `padding-bottom: env(safe-area-inset-bottom)`。

## 脚本逻辑
- 新增状态：`uiMode: 'list' | 'reader'`；函数 `setMode(mode)` 添加/移除 `body` 上的模式类并控制各区域显示。
- 初始化：解析 hash，决定模式与加载文章；记录并恢复列表滚动位置。
- `openPost(index)`：设置 `uiMode='reader'`，写入 `#post/<filename>`，渲染正文与目录锚点，滚动置顶；更新分页卡片标题状态与禁用态；预取相邻文章提升切换速度。
- 返回按钮：`mobile-back` 点击时 `setMode('list')`，写入 `#list`，恢复列表滚动。
- 分页点击：`pager-prev`/`pager-next` 分别加载相邻文章并更新 hash。
- 密码逻辑：列表可见；阅读时校验密码，未授权弹出密码卡片。

## 变更文件

- `lazyblog/site/style.css`：新增移动端模式类与分页样式；隐藏/显示对应区域；隐藏浮层目录与“列表”按钮于阅读模式。
