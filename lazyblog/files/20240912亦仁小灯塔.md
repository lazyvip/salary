# 01 如何平衡好主业和副业？ 

#小灯塔  01

这个问题，每一年，都有好多圈友在星球问。也有不同阶段圈友的不同回答，我也回答过这个问题好多次，答案在不断改变。

现在，我对这个问题看法如下：

1/ 你一定要搞清楚一件事：你的主业是在为你的副业做准备，你的副业是在为你的事业做准备。尽量把主业和副业看成是一件事，不是去平衡，而是去衔接。

2/ 我强烈认为，每个人，都要有自己的事业，或早或晚。

3/ 有一件事，可以让你全身心 100% 心无旁骛投入进去，哪怕失败，又何尝不是一种全新的独特体验？人生在世，就是为了获得足够多不同体验。

4/ 我说主业是为副业做准备，并不是说让你在主业里摸鱼，恰恰相反，你应该在主业上像创业一样疯狂投入和尝试。

5/ 因为这是一个赔率巨大的事情：尝试成功了，碰到不算太差的老板，会给很多回报和机会。尝试失败了，学到了经验和技能，有老板兜底，也不至于破产亏钱负债。

6/ 在主业中尽量学习副业中最需要的能力：如何从0到1，其中更重要的是如何进行低成本尝试，这是我认为赚钱最重要的能力。

7/ 如果你能近距离接触老板或者操盘手，看他们是如何从0到1做成一个业务或者一件新事情，那你是幸运的，请认真学习。哪怕经常被骂。

8/ 赚到钱，核心就是因为控制住风险和成本，让自己不要出局。只要不出局，就可以不断尝试，直到成功。

9/ 如果副业已有起色，但考虑到收入问题还舍不得立刻放弃主业，请把副业收入的一部分拿出来招线上助理，不要舍不得，哪怕你空出时间来休息养精蓄锐，或者陪陪家人。

10/ 助理最好从同学、家人或者朋友开始，他们会更相信你。因为这个时候你根本不知道怎么从 boss 直聘上，大多数人也会把你当骗子来处理。

11/ 但你最好有两个预期：第一，你的助理并不会帮你干的多好，你会觉得自己亏了，这个钱花的不值；第二，如果你的副业很赚钱，你的助理最后很可能会变成你的竞争对手。这些没关系，大家都经历过，先解决眼前的问题，未来的问题未来再说。用资金换时间，用时间换空间。

12/ 关键是做好取舍，不可能副业也好，主业也好，休息娱乐一个不少。我还没见到谁能做到，赚到大钱的人，背后基本都有巨大付出，只是有些付出不是你能想象到的维度。

13/ 当我们问如何平衡主业和副业时，是不是在问如何保持休息和娱乐时间都不少的情况下，副业和主业都能获得增长？想什么呢，凭啥是你？

14/ 和老婆商量下，周末挑一天出来，出去找个付费自习室，拼命干一天，基本等于你每天下班回来干一小时干半个月的效果。记住了，及时的把一些进展分享给你老婆，未来你可能获得两天的全职副业日。

15/ 不要长期熬夜，保重好身体。

16/ 争取副业干三年赚100万，让副业光荣转正。

17/ 加油，来到生财有术，你已经赢在了起跑线。

# 02 怎么熬过没有正反馈的低谷期？

#小灯塔  02 

很多圈友问我，我也参加航海了，我也按照航海手册行动了，但在 21 天内，我不像一些圈友那样取得结果，甚至我连反馈都没有，我该怎么熬过没有正反馈的低谷期？

很好的问题，我的看法如下：

1/ 首先，这不叫低谷期，这叫常态。我们做的大多数事情，没有正反馈是正常的，有负反馈也是正常的。

2/ 对我而言，有负反馈是好事，停止做错的事情本身就是前进。

3/ 大多数情况下，我们面临的其实是没有反馈。所以，做一件事之前，尽量先问自己，这个事情的反馈是什么？需要很长时间才能看到反馈吗？能不能向下拆分一下，让反馈可以更快出现？

比如，我想做小红书，不把在小红书赚到钱定为目标，而是把第一篇笔记顺利发出，第一篇 1000 曝光的笔记，第一个赞，第一条评论，第一个粉丝，第一个引流到微信，第一个转化，第一个用户好评，等等这些定为反馈节点。

4/ 在赚钱这件事上，我算身经百战。但我现在还在经常失败，包括生财有术，我做了很多很多尝试，被大家看到的只是少数，大多数想法和尝试都以失败告终。

我看到很多很厉害的人，也在经历很多失败，一个人失败很少，只能说明一件事：他没有在做新事。

5/ 我在修炼一个对于创业者很重要的能力：如何平和面对负反馈以及没有反馈。

当我有一天可以全然投入到我认为重要的事情，不再为失败而感到烦恼，不再被别人的看法影响情绪，说明我可能又进化了。

我很幸运的获得了影响力，这影响力又变成一种束缚，如何从这里面走出来，自在的表达和创作，这是我正在修炼的。

6/ 成事者，生财之术固然重要，但更需要的是强大的心力。心力来源于对自己信念的强烈聚焦。多问问自己的信念是什么，写下来，经常看看，也许更容易熬过没有反馈的低谷期。

7/ 多用别人的成绩和案例来激励自己，大家都是九年义务教育，凭啥你行我不行？在这个角度下，对于那些晒赚钱结果的人，宁可信其有，欢迎您多晒。

8/ 坚定一个信念：我们大多数普通人赚的钱，基本都是抄作业。抄作业说明此题确定有解，开卷考试，那解题难度远远小于创新，创新难在你不知道自己做的题目是不是有解。

但你一定知道，从小红书一定是可以搞到大量流量到私域的，这道题 100% 一定确定以及肯定有解，暂时搞不定那只是方法问题，反复去拆解研究尝试请教，一定可以找到办法。

9/ 远离负能量的人，包括根本不相信能靠自己赚到大钱的人、觉得所有知识付费都是骗局的人，多与积极的、乐观的、向上的、务实的、真诚的、利他的、开放的人在一起，人和人之间会互相影响。

10/ 尽量多积极主动把自己做的事情拿出来讲讲，听听比你厉害的人的反馈。他们的一句话，你认真听进去，可能会改变一辈子。

11/ 在生财有术，展现一个真诚的、利他的、有结果的自己，是吸引战友非常高效的方法。想象一下，如果未来刘小排创业找合伙人或者员工，一定会非常容易，排队抢着来。

12/ 尝试找到一个能理解你、支持你的战友，一起互相激励共同前行，就不那么容易感到孤单和绝望了。

13/ 手机上装一个豆包软件，做一个赚钱鼓励师的智能体，遇到任何困难和负面情绪的时候，找它说说，甚至骂骂它，它可能会帮你走出来。

14/ 赚到钱，无非就是无数个微小里程碑叠加后自然而然的结果。

15/ 有一些小进展就奖励一下自己，比如吃点好吃的，或者买一个新款手机。经常庆功，就能成功。再微小的成功，都值得庆祝。

# 03 如何对听到的各种赚钱信息做出有效判断？

#小灯塔  03

你们放心，我断不会让小灯塔变成月更。来，咱直接开始。

1/ 两个默认：默认项目都是通的，默认数据都是假的。如果有人分享了一个赚钱项目和收入数据截图，请按上面的原则来判断。

对项目乐观，可以让自己去了解细节。对收入数据悲观，可以让自己谨慎投入。

2/ 亲自去体验感受细节是最重要的，只有自己亲自思考过、验证过才决定是否投入。

当别人跟你说一个新的赚钱机会时，别高兴太早，几乎所有新的“赚钱机会”、新的“蓝海市场”，走近仔细看时，会发现已经有很多竞争对手在那边，甚至会发现，他们在那边已经闷声赚了好多年的钱。

没有红利，只有实力。实力到了，都是红利。

3/ 绝对不投入做任何无法亲自体验和感受的项目，就像绝对不买任何你不亲自使用产品的公司的股票一样。

别轻易冲进一个新的所谓赚钱的市场，谋定而后动，调研清楚机会、能力、团队，再动也不迟，不追求第一波吃螃蟹的人，市场足够大，第二波、第三波也可以，这样可以少踩很多坑，这就是敢为天下后。

别担心错过一个好机会，如果你觉得你可能在错过一个好机会，说明这机会根本不属于你。每时每刻，都有很好的赚钱机会存在。就像芒格说的，当你的认知和执行配得上时，好机会自然会来。

当下，如此艰难的时刻，我仍然知道有圈友一个月能赚千万甚至更多，所以，好机会永远在。

4/ 我理解的赚钱 = 有效决策+有效执行，有效决策 = 优质信息+思维方式，有效执行 = 假设驱动 + 高效试错。

信息是起点，但优质信息只是赚到钱的一小部分。即使同样的信息，执行也会天差地别。项目做不起来，先从自己身上找原因。

捕捉各种信息点要足够敏锐，别人不会毫无保留分享赚钱信息，但说话中会有各种零碎的线索，带着线索去研究，可以发现很多机会，很多赚钱的业务就是在一两个关键词带出来的，背后是各种信息挖掘能力。

5/ 这个世界上的信息 90% 以上都是噪音，真正有价值的信息非常少。希望屏蔽噪音，需要额外付出成本，比如金钱或者时间。进了生财有术，你已经屏蔽掉抖音等平台更大的噪音，但生财有术也有噪音。

6/ 为什么生财有术这样的付费社群也有噪音？人多的地方就是这样。我在努力给社区降噪，但也需要平衡好：用力过猛，可能会导致没有声音或者只有一种声音，那更可怕。

7/ 我加过的最贵付费社群 13万/年，连续续费多年。我推荐了一个朋友加入，朋友加入一年后退出，说噪音太多。

核心是你知道自己想要什么信息，你知道自己要什么就能快速找到重点。如果不知道，那有价值的信息出现在你面前时，也会被识别为噪音。

8/  最有价值的信息都在人身上，如果不知道如何找到有价值的信息，那先跟随你觉得厉害的人，或者与这些人保持联系。

但你得能区分：谁是真正厉害的，而谁又是在虚张声势。我的区分方法很简单：我只相信拿到可公开验证结果的实战派。

一个连团队都没有的人，我是绝对不会去听他的任何管理课程。

9/ 超过10个人的聚会基本就不会有啥太核心的信息说出来，特别是有陌生人在场的聚会，参加大型聚会的目的，很多时候是为了可以高效找到愿意小圈子分享的人。

但更核心的，是你得有能拿出来交换的信息或价值。认真想想，是什么？

10/ 轻松赚大钱的方法，不一定写在刑法上，但一定不会在一开始，就出现在互联网上，更不会出现在公开场合的分享中。有时候，我在小圈子中听到的一些很好的赚钱机会，去微信搜朋友圈和微信会话，很惊讶的发现，搜索不到任何消息，要知道，我微信上可是 10000 位对各种赚钱机会敏感的人。

11/ 要有赚钱信息有判断力，核心是赚到过钱，而不是很多人以为的：赚到钱，核心是对赚钱信息得先有判断力。

刚开始赚到钱，都是瞎猫碰到死耗子，撞上了，然后才有了一些基础判断力，如此循环向上。

所以，核心就四个字：先干起来！如果要在前面加三个字，那就是：低成本先干起来！

12/ 如何低成本？除了可能的学费外，凡是要先投钱的一律不干。很多人从来没做过项目，手里好不容易辛辛苦苦攒了一笔钱，想投资个项目靠钱来生钱。不好意思，通过投资赚钱的事情，轮不到你，甚至也轮不到我。有很多老板排着队想给我钱，让我有好项目带他们投投。你说，好项目会轮到你和我那几十万吗？

默认所有要你先投钱的项目都是骗子。当下，在起步阶段，核心一定是投入你的时间和精力，摸索出如何做项目赚钱的经验。

15/ 大多数情况下，如果你一直在职场路线，你想搞个副业或者创业，请空杯心态从零开始。你的职场经验大多数时候会对创业赚钱起反作用，职场需要服从和执行，而创业需要不断的反向思维和大胆突破。

16/ 当你真正理解了以上，弄懂了如何判断信息，恭喜你，所有的平台都是你的优质项目库，你会真正读懂这首诗：

众里寻他千百度，蓦然回首，那人却在，灯火阑珊处。

# 04 如果财富归零，我会去哪里寻找好的赚钱机会？


<span>#小灯塔#</span> 04

今天这篇，讲具体方法，我会告诉你，如果财富归零，要去寻找好的赚钱机会，我会去哪里找？

在开始之前，先讲几条核心底层逻辑，希望我们能够有共识，因为这些逻辑影响着如何选择机会。

1/ 正确认知自己，你只是想赚钱还是想做一个伟大的产品改变世界。目标不一样，做事方法不一样。对于大多数圈友来说，多赚点钱改变个人和家庭生活，应当是主要目标。

2/ 想赚更多钱，一点都不丢人，在生财有术不想赚更多钱才丢人。抛弃掉大多数中国人骨子里的赚钱羞耻感，是赚到钱的第一步。

3/ 作为大多数想赚钱的普通人，10人以下团队，包括个体，核心工作都应该是搞流量，搞流量，搞流量，重要的事情说三遍。

赚钱就是搞流量，搞流量就是赚钱。当你能力和资金不足，却开始搞自己的产品，大概率走向了失败和亏钱。

4/ 在目前这个时代，搞流量基本等于在大平台规模化搞内容（包括短视频和直播），搞内容的核心是找到爆款然后微调，爆过的内容一定会再爆。

5/ 找一个可以卖的产品很容易，做出一个产品也很容易，最难的是把产品卖出去，也就是跑通项目：流量-转化闭环验证。

起步先把最不确定最难干的事情验证好：先搞流量，再卖别人的产品，最后才是自己做产品。什么时候开始自己做产品？靠流量赚到第一桶金后，比如100万。

6/ 我寻找赚钱机会，一定不会直接做从生财有术精华帖或者风向标里看到的项目，大家都盯着的项目肯定不是好项目。

但我一定会从某一个风向标或者精华帖里去观察背后趋势，然后深入挖掘，充分拆解，举一反十，找到一个潜在市场不小、竞争不高的机会。

7/ 在内容平台，每一个持续更新的账号等于一个赚钱的项目，每一个投放的广告等于一个正在放大的赚钱项目。仔细研究，ta是怎么赚钱的？

8/ 从零碎的情报线索，比如风向标，到一个清晰的赚钱机会，中间是深入拆解和仔细算账。
