# 整合：成年人的反向观察指标

> 作者：和菜头
>

整合，动词，意思是集结不同的意见或事物，重新统合，成为新的整体。

*

一个概念和它对应的现实可能会发生背离，整合就是其中的一个。整合当然是个好词，听起来积极向上，未来一片光明，但是在现实中我极少观察到成功的整合，尤其是在大公司里。每当这个词出现时，往往意味着事情已经无法收拾，才需要自上而下所谓 “整合” 一下。于是，

**整合的结果和通常和折腾相差不远。**

通过观察任何一个概念，和这个概念的实践，我们就可以借此观察社会，观察他人。比如说诚实，它是个很好的概念，而且有道德色彩的加持。人们从小到大受到的教育都在强调诚实，但是如果在某个地方，人们发现有的人会因为诚实而遭受惩罚，有的人会因为欺骗而大获其利，那么诚实这个概念在面对现实的时候就崩解了。而且不单是诚实这个概念，连带美德也一并受到了质疑，人们会很清楚地知道，自己身处一个不诚实的环境，在这样的环境里保持诚实无异于自虐，这就是观察的结果。

整合本来也是一个很好的中文词。但是就我个人的观察，起码在我这一代以前的中国人，天性中更倾向于自主自立，喜欢扮演主角，一般不会考虑做配角打配合。

任何一个人，如果给他一小块领域，给予他足够的权限，那么他会在他的一亩三分地里干得很欢快，而且出成绩很快。

但是如果你把十个这样人的召集起来，让他们彼此支援，取长补短，共同完成一件事情，那么结果多半都不如人意。

有个很形象的比喻，叫作 “螃蟹篓子”。据说捕蟹人的蟹篓从来不需要盖子，因为单独的一只螃蟹会很快从蟹篓里爬出来，但是如果有许多只蟹挤在一个蟹篓里，那么任何一只螃蟹试图爬上去的时候，其他螃蟹也会做同样的事情，结果就是它们每次都能成功地用蟹爪把那个正在攀爬的“兄弟” 给拽下来。螃蟹永远都不会进化出合作意识，也永远不会进化出顺序攀爬的想法，所以螃蟹篓子也就不需要盖子。

现实中的整合为什么经常不成功？为什么资源和人力不能统合起来，发挥更大的效用？为什么人们总是倾向于各自为政，不得不合作时则相互防备、相互掣肘？遇见这种问题，有一种偷懒的办法，那就是解释为国民性，认为这是国民的某种特性，甚至归结为所谓劣根性。我不那么看，我认为

**这是一切人类组织的共同特性。**

观察任何一个组织的发展，会发现组织的复杂化会是一种总体趋势。组织的每一个分支都会通过拓展业绩的方式发展出更多新的业务，因此提升自己的独特性。这是因为如果每个分支都在做类似的事情，那么它们就可以彼此替代，这样就会降低了自己的存在价值。还有一个因素不容忽视，那就是在一个组织内部，总存在资源分配的问题。如果某个分支从事的业务越多越复杂，意味着可以分配到更多资源，于是这个分支的生存状况就会好很多，分支成员也就有更多行动的意愿。

在复杂化的同时，每一个分支又都有形成自给自足的封闭体系的趋势。不单是某一组织的分支如此，组织本身也有这样的趋势。我们在生活中经常可以观察到，一家公司一旦做大做强，就会抑制不住地产生控制上下游的冲动，对此还有个很好听的名字，叫作 “全链控制”。为什么会有形成自给自足封闭体系的趋势？因为合作是不可靠的，合作是不可信的，合作是不可控的。假设一家公司有一个 IT 部门，当所有部门都向 IT 部门提出需求的时候，那就不得不排队等待。等得久了，大型或者强力部门就会自己组建单独的 IT 小组，这样就不需要等排期。像这样发展下去，每一个部门都会变成小而全，尽量避免和其他部门打交道。

有这两个趋势存在，整合就变得非常困难。因为整合完全违背了组织内各个分支的生物本能，也危害到了它们各自的利益。

**这当然不是什么国民性造成的结果，而是一个信任问题。**

在多大程度上可以信任合作这件事，在多大程度上可以信任合作方，在多大程度上可以信任双方之间的契约。从反面来说，那就是在一个社会里，一方背弃合作究竟会受到多大力度的惩罚？如果这种惩罚是严厉的，不单合作伙伴会中断业务往来，全行业还会集体摈弃，那么信任就得到了保证，合作就是可能的，可以放心地做单一业务，可以放心地把一部分业务环节移交给合作方。如果不是，那最理性的原则还是让业务复杂化，让自己部门的功能完备化，这样就不需要受制于人。

在中国传统社会的生活图景中，大部分人都是农人。农人有田地，田地能解决口粮问题。门口有菜地，菜地能解决蔬菜问题。院子里养猪、养牛、养鸡，能解决肉蛋和畜力问题。农人的太太和女儿会织布和缝纫，自纺布能解决穿衣问题。在这样的生活图景中，一户人家的生活完全可以靠自家的力量解决，不大需要社会生产和社会合作介入。但也正因为这样，一户人家可以在很长的时间里自立自全，在哪里都可以存活下去。

我们可以想象，如果日用品的商路通畅稳定，货品质量稳定可靠，价格公允少波动，那么这户农人完全可以不用从事那么多的活计，可能专心种地或者专心养蚕就够了。问题在于，他们在多大程度上可以信任商人和商路，信任远处某个陌生人的产出？

从信任自家人，到信任同族人，再到相信同乡人，最后到相信陌生人，中间要完成很多次跨越。

但更为重要的是，人们需要让自己的信任得到确认和保护，建立对社会生产和对陌生人的信心。也就是说，

人们需要有规则，而且在现实生活中确信规则会得到遵守。

整合不成功，往往是因为缺少规则，或者即便有规则也得不到尊重。没有规则，也就无法形成秩序。人只有在秩序之下，才会各安其位，各司其职。

所以，在现实的世界里，当一家公司开始要整合业务的时候，只会有两个原因。一个原因是老板强力推行，他要强行建立一套新秩序，所有人不得不执行；另一个原因是山穷水尽，所有人都受到了威胁，整合是最后的救命稻草不得不紧紧抓住，依附在新秩序下。但不会是因为整合是对的，也不会是因为整合对所有人都有利。

因此，

**一个成年人要选择定居的城市，要选择投身的行业，要选择应聘的组织或者公司，首先应该观察这里的有没有规则，有没有秩序。**

如果规则高于人，秩序高于人的意志，那么大可以去得，无论是生活还是工作，在规则和秩序之下，都省心省力，更接近于一个现代社会应该有的样子。如果缺乏依据，那么

整合是一个很好的反向指标，当整合这个概念反复出现，就说明规则在这里得不到尊重，每个人都在做自给自足的农人，机会和风险同样巨大。
