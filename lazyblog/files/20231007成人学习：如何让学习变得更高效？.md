#  成人学习：如何让学习变得更高效？

成人学习，名词，意思是成人为增进知识、获得技能、改变态度或者自我完善而进行的学习活动。包括有组织或无组织的学习、正规或非正规的学习、他人导向或自我导向的学习等。

*

朋友说我是那种在成年后少见的能依然保持浓厚好奇心的人，这话不全对。我自己很清楚地知道，我的好奇心也随着年岁不断消磨，和所有人一样，年岁越长，好奇心越少。真正的区别在于，在那些少数还能激发我好奇心的事物上，我拥有对应的学习能力，成年人的学习能力。

这里需要谈到 “成人学习” 这个概念。

颇有一些人认为，学习和学校教育是一回事，出了学校之后就是混社会。但是我自己的看法不同。我认为，和人们漫长的一生相比，学校教育只占学习的很小一部分，而且可能是最无趣的那一部分。

人的大部分教育都来自成年之后的学习，而且只有这一阶段的学习才算得上是真正的学习。

我此时此刻的生存技能，我的人际交往方式，我个人对生活和生命的理解，都学自成年之后。中小学教育知识提供了基础，大学教育提供了思考方式，但真正学到一点什么，都发生在离开学校之后。

最伟大的教师只有两位，一位叫作兴趣，另外一位叫作实践。

它们的教学成果不是分数和学位，而是对于个人而言最宝贵的礼物——成为自我。

学校里从未教过我关于互联网的知识，从未教过我如何创造互联网产品，也从未教过我写文章，更不曾教我使用哪一种文字才能最好地表达自我。所有这一切，都是我在成年之后自己学会的，而且都是自学的结果。于是，我才能成为互联网上的那个和菜头，而不是建筑业的和菜头，或者文化界的和菜头。这刚好也是我想要的，我喜欢互联网，我觉得互联网就是我的花园。

在这个过程中，我觉得成人学习有几个特点：

首先，它有不利的一方面，那就是脱离了学校教育的大环境之后，学习变得困难许多。

学童们精力旺盛，心思单纯，每天除了学习之外没有多少需要操心的事，而成人不是这样。成人很容易陷入无穷无尽的生活琐事中去，心思散乱，随时都处在精疲力竭的状态，想要找出一点时间来学习，要比学生时代困难得多。

然后，成人比学童更加畏难。

成人有了经验，不再是不知深浅、可以一头扎下去的学童。所以，即便有了决心，有了时间去学习，也很容易在预估一下难度之后就决定放弃。因为成人会计算自己的时间成本，所有需要付出大量时间但是短期内看不到成效的事情，他们都会非常谨慎小心，更倾向于放弃，而不是坚持。

最后，影响成人学习的诸多因素中，个人意愿甚至要比学习能力更为重要。

单位组织的学习培训，通常都是一种灾难，因为参与者缺乏真实的意愿，也就没有任何动力。但是个人想要改善处境，提升职业能力的时候，不需要任何人安排，成人会自行报班，自行购买教材，自行安排时间，主动去学习，为的就是得到一张司法考试或者会计师考试通过证明。

不过，成人学习有一项最大的优势，那就是实践。

对成人而言，对学习最有利的支持就是实践。

只要一个成年人是为了在生活中解决某一个现实问题，那么他就能够学习，而且学习的速度会很快，因为可以学以致用。

我们通常发现，有些成人决定要学习一门外语，或者一门计算机语言，往往都不会成功。他们有真实意愿，也有真实付出，但是就是不能学成，唯有那些大决心、大毅力之辈，才有一线希望。

为什么？

因为我想学一门外语，我想学一门计算机语言，这是学校教育时代才有的想法。这种想法对应着学校，对应着课表，对应着教师和教室，以及从零开始的系列进阶教材和大量的作业和考试。这是学生学习的方式。

但是对于成人而言，已经没有学校和教室了，没有人给你安排课时，没有人用作业和测验来反复训练你，用成绩和排名来督促你。整体的学校教育环境已经消失，靠个人的力量在自己家里复制出一间教室来是没有可能的。

但是，对于一个成人而言，如果仅仅只是为了不看字幕就能看懂剧目而学习一门外语，这就是有可能的。因为他不需要教科书，他只需要一点点记住电视屏幕上的口语表达，下次听到时能够认出来就足够了。于是日积月累，他居然也就一点点学会了一门外语，未必能说，但是会逐渐能听懂。

如果他是想为自己编写一个程序，解决生活或者工作中一件恼人的事，甚至只是为了给孩子设计一个简单的电脑游戏，那么他多半也可能会成功。因为他明确地知道自己想要什么，需要学什么，然后学来的东西能实现什么。他一开始可能完全不知道原理，但是他知道能实现什么也就足够了。随着信心和经验的增加，他最终就能掌握一门计算机语言，甚至还不止于此。

实践能证明学习有效，实践能最真实地反馈学习成果，实践也能为学习指明方向，所以成人的学习可以是高效而迅速的，远远超过任何学生，这是成人学习最有利的地方。

就我个人的感受而言，成人学习完全是另外一种东西。大多时候，一开始没有什么体系，也没有什么基础学习，更不存在什么学到某种程度之后具备解决问题的能力。成人学习没有这些准备时间，也不会那么从容不迫，它上来就是直奔问题而去，追求的不是知识，而是解决问题的方法。在解决了足够多的问题之后，学到的知识和经验会自动形成体系，和教科书或者教师授课完全不同。

只要在心头放下一些过往形成的执念，成人学习就可以变得顺利许多。

比如说，放下教材。把教材从入门读到精通，你看到的也不过是一些字而已，而且还不是你的。你能解决一个问题，然后再在教科书里翻到有关的那一页证实，那一页记载的内容才是你的。

再比如说，放弃完备和完美。任何时候你想出来的方法都不可能是完备的，更不可能是完美的，你只是简单地想到了一个方法，粗糙地搭建了一个工具，最后部分解决了问题。但这一点都不重要，如果你一开始就追求完备和完美，那么你就永远都不能完成，无法完成你就没有从中学到任何东西。

成人学习有很多功利的部分，有很多粗糙的部分，有很多省略的部分，但是为了换取最后的可用，这一切都是值得的。

学生学习是把一本书从头读到尾，成人学习是一边做事，一边撕书，做事需要用到哪一页上的知识，那么就撕下来放在手边赶紧学，学完了抬头就开始试着操作。学生的书是用一学期时间逐次读完的，而成人的书是在完成一件件事的过程中一页页撕完的。读完的书有一大部分要还给老师，但是撕完了书都全部归于自己。

这就是我对成人学习的理解。我不认为我的好奇心比人强，但是我认为，我有足够的勇气去撕书。

这样，当人们同样面对一件新鲜事的时候：有些人会计算和分析，掌握和运用它需要多少知识，准备多少本书，拥有多少知识和信息储备，然后就犹豫不前。而我一旦下定了决心，就会立即上手，先拿它做点我想要的东西再说。

所以，最后看起来，是我的好奇心支持我向前探索了很长一段，但我自己清楚，这是因为成人学习的方式——先往前推，推不动了再来学习，那么你就能真正学到点什么。这也是我觉得自己比在学校里学习更为快乐、更觉有趣的原因。
